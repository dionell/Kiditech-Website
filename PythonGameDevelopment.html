Python coding with game

GRADE REQUIREMENT:

Grade 5+

COURSE DESCRIPTION:

Come see why Python’s simple and readable code is easy to jump into, but powerful enough to be used by Google and NASA. You’ll learn the ropes of this object-oriented programming language by creating your own game. Import 2D graphics and sound libraries to make your project even more unique. Take home your game to play with friends and show off your new coding skills.

HARDWARE and SOFTWARE PLATFORM

Hardware: Raspberry pi 3 B+ Board and other DIY kits like video car kit

Software: Linux system Raspbian and python programing

LEARNING OUTCOMES:

  • Create variables and function
  • Control flow including if and loop
  • Use object oriented design principles
  • Program in IDE
  • Get user input manipulating graphics
