Game Design with Unity


GRADE REQUIREMENT:

Grade 5+

COURSE DESCRIPTION:

Design and develop your very own game with the power of Unity. It’s the most popular and professional tool among game developers, so you’ll be building skills that you can take to college and beyond. You’ll also get a chance to hone your programming skills using C# to bring your creation to life. Delve into all aspects of game creation with a full development pipeline: implementing art, sound, and scripts to complete a project you can be proud of and take home.

LEARNING OUTCOMES:

  • How to re-use game element from assest store in Unity
  • Understand Fundamentals of the Unity game engine
  • Build a basic 3D supermario clone game with multiple game level
  • Learn how to create 3D models and animation in Unity with C# language
  • Create your own 3D game
