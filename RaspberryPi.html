IoT with Raspberry pi

GRADE REQUIREMENT:

Grade 6+

COURSE DESCRIPTION:

The Raspberry Pi is a series of small single-board computers developed in the United Kingdom by the Raspberry Pi Foundation to promote the teaching of basic computer science in schools and in developing countries, it is open source hardware function like computer with GPIO pins and wifi chip, which enable IoT application. During this course, we will learn how to make a Linux computer with Raspberry pi , get familiar with Linux system with basic linux commands, text program in python for video car motion control, video stream and piture capture, design graphic user interface for car remote, learn TCP/IP and socket programing, talk to twitter and google voice assistant with API.

HARDWARE and SOFTWARE PLATFORM

Hardware: Raspberry pi 3 B+ Board and other DIY kits like video car kit

Software: Linux system Raspbian and python programing

LEARNING OUTCOMES:

  • Learn Linux system and master basic linux command.
  • Learn Python and socket programing.
  • Complete interesting hands-on project like twitter pusher, Video Robot car and google voice assitant.
  • Collaborative teamwork for multiple level challenge
  • Build a complete project from scratch.
  • Team work and pitch.
